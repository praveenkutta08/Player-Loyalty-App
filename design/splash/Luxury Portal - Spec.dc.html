Luxury Portal

Visual design. Standing before an entrance at dusk: a warm light grows somewhere beyond, then a hairline circular aperture dilates — an architectural opening, like a rotunda oculus or a grand doorway — and the brand is revealed through it by a circular mask. A second, quieter ring settles outward like the molding of the doorway. Two hairline circles, one light, one mask — nothing science-fiction, nothing more.

Design rationale. Every great resort choreographs the moment you pass its threshold: light beyond, an opening, then the reveal. Portal compresses that sequence — arrival → discovery → welcome — into 2.4 seconds. The circular geometry reads as exclusivity (a private entrance, a member's seal) while staying minimal: it is the only variant whose reveal is a true mask, so the logo appears through something rather than fading in.

02

Portal geometry

Shared layout system — safe areas 44 / 24 / 34 dp, emblem box 96 dp at 50% H, wordmark block at 60% H. The portal layer adds three concentric elements, all centered on the emblem:

interior light · Ø 52% W circle · radial primary@44% → transparent 72% · the "beyond"
aperture ring · Ø 46% W · 1.5 dp hairline, primary@85% · glow 0 0 18 primary@30% outside + @16% inside
outer frame · Ø 46% W → ×1.22 · 1 dp hairline, primary@45% · rests at 25% opacity
reveal mask · circle 0 → 75% on the emblem — the logo appears through the opening, never a plain fade
Ring diameter (46%) deliberately exceeds the emblem box (27%) — generous negative space inside the opening reads as exclusivity, not a badge.

06

Accessibility

Reduced motion — aperture and mask are skipped; 300ms cross-fade of the settled frame (ring at rest, emblem revealed), then navigate. Shared code path.
Contrast — name text vs ground ≥ 4.5:1 validated at ingest; hairline rings are decorative (not information-bearing) so their 25–90% opacities carry no contrast requirement.
Vestibular safety — the aperture dilates once, slowly (≈0.6s); no rotation, no oscillation, no zoom-through-the-portal camera move. Well inside WCAG 2.3.3 comfort.
Screen readers — root announces "‹Casino name›. Loading."; hand-off announces Home. No focusable elements inside the splash.
Time limit — hand-off at timeline end or when Home is ready, hard cap 4s.
07

React Native · performance · handoff

Rings — two plain Views with borderRadius 50% + hairline border, animated with scale/opacity worklets off the shared clock. The glow is a static shadow style, not an animated blur.
Reveal mask — @react-native-masked-view/masked-view with an animated circle (or a Skia circle clip if Skia is in the stack). Fallback for low-end devices: opacity + scale 1.05→1.00 — visually close, mask-free.
Performance — 4 animated layers total (light, 2 rings, emblem), transform/opacity only; the single blur (4→0) is on the small emblem via pre-blurred cross-fade. No runtime blurs, no particles, no full-screen effects. Budget: 60fps on Android Go-class hardware.
Hand-off — ring and emblem shrink together to the header slot (scale → 0.42); the ring fades during flight so only the logo lands. Ground persists into Home.
