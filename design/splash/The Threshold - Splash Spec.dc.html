Selected concept · Production spec

The Threshold

A single luminous horizon parts the dark; the brand emblem rises through it and settles as the wordmark resolves beneath. Not a logo dropped in the middle — an arrival. The horizon is the signature; everything else is a CMS token flowing through it.

Screen layout

Vertically anchored to a single horizon at 50% H — the optical centre. The emblem is centred on that line so it reads as rising through it; the wordmark block hangs in the lower third at 60% H. Nothing is bottom-pinned, keeping the composition brand-agnostic and calm.

Light & dark mode

Dark · default
Light · daytime

Same geometry. In light mode the ground lifts to a warm paper tone, text flips to ink, and the horizon & bloom hold the brand hue — never pure white. Chosen from system appearance or a CMS override.

Design system

Typography

MANROPE
Tagline & system UI · +0.34em uppercase
Color roles
primary · emblem, horizon, bloom, tagline
background · CMS gradient / image
text · ink or cream by mode
Gradient usage

Two-stop vertical ground (178°) + one radial bloom rising from the horizon in primary @ ~55% + a top scrim for status legibility. Never more than two ground stops.

Shadow usage

Flat by design — no drop shadows on text or emblem. Depth comes only from the horizon glow (0 0 26 primary@72%) and the bloom. Elevation is reserved for the app.
